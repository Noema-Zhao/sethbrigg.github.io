hugo建站-Windows

May 4, 2022 · 1 min · Noema

  • 2. 生成站点

Hugo是用 Go 语言编写的跨平台静态网站生成器，简单易用门槛低。
以下是个人部落格在Windows上的部署过程，供参考。

1. 安装hugo#

下载网址： https://github.com/gohugoio/hugo/releases 选择：

hugo_extended_0.98.0_Windows-64bit.zip

环境变量中添加hugo文件路径，如：

e:\hugo\hugo.exe

2. 生成站点#

*以下代码均在powershell中实现：

cd e:\hugo
hugo new site blog

💡: Congratulations! Your new Hugo site is created in E:\hugo\blog.

创建文件夹为blog，以下内容均在blog中实现： cd .\blog\

3. 选择主题#

主题大卖场位于： https://themes.gohugo.io/

本部落格选择Papermod主题🖊

git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1

打开config.toml,添加： theme = 'PaperMod'

4. 新建文章#

hugo new post/article1.md

打开 ./content/post/article1.md:

可以修改标题，如改为:

title: “hugo建站-Windows”

写入文章（markdown格式）。

草稿完成修改字段:

draft: false

5. 本地调试#

hugo server -D

💡 : Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)

进入1313网址，可以看到我们的第一篇小文章啦（要是发paper能有这么容易就好了（怨念…

1313酱会实时变化，记录更新。

这时可以根据主题的介绍页修改config.toml文件，完善部落格的生态系统🌳

PaperMod介绍页

6. 部署到GitHub#

在GitHub上建库，名称为：[yourusername].github.io 例如我的库叫做： sethbrigg.github.io

hugo --baseUrl="https://sethbrigg.github.io/" --buildDrafts

我们的站就被放在public路径下啦~

cd .\public\
git init
git add .
git commit -m "hugo 1st commit"
git remote add origin https://github.com/SethBrigg/sethbrigg.github.io.git
git push -u origin master

这时其他人就可以在 https://sethbrigg.github.io/ 看到我们的blog啦~ 🌸🌸🌸
